# 🎮 Jogo Desenvolvido na Unity

Jogo totalmente funcional desenvolvido utilizando a Unity Engine e linguagem C#.

---

## 🎯 Mecânicas Implementadas

- Sistema de movimentação do personagem
- Sistema de colisão
- Sistema de pontuação
- Controle de vidas
- Interface gráfica (UI)
- Reinício de fase
- Condições de vitória e derrota

---

## 🛠 Tecnologias Utilizadas

- Unity Engine
- C#
- Física 2D/3D
- GameObjects e Scripts
- Sistema de Input

---

## 🧠 Conceitos Aplicados

- Programação orientada a objetos
- Eventos e colisões
- Controle de estados do jogo
- Estruturação de cenas
- Organização de scripts

---

## 🎯 Objetivo do Projeto

Aplicar conceitos de desenvolvimento de jogos, lógica de programação e design interativo.

---

## 📚 Aprendizados

- Estruturação de projetos na Unity
- Organização de scripts em C#
- Implementação de mecânicas jogáveis
- Debug e testes em ambiente de game engine
