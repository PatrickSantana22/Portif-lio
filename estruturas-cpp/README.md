# 🧩 Estruturas de Dados em C++

Projeto desenvolvido com o objetivo de implementar manualmente estruturas de dados fundamentais utilizando C++, reforçando conceitos de ponteiros, alocação dinâmica e complexidade de algoritmos.

---

## 📚 Estruturas Implementadas

- Lista encadeada simples
- Lista duplamente encadeada
- Pilha (Stack)
- Fila (Queue)
- Operações de inserção
- Remoção de elementos
- Busca
- Impressão da estrutura

---

## ⚙ Funcionalidades

- Inserção no início e no final
- Remoção por valor
- Busca sequencial
- Controle de memória com `new` e `delete`
- Manipulação de ponteiros
- Testes via menu interativo (terminal)

---

## 🛠 Tecnologias Utilizadas

- C++
- Programação estruturada
- Manipulação de ponteiros
- Alocação dinâmica de memória
- Compilação via g++

---

## 🧠 Conceitos Aplicados

- Estrutura de dados linear
- Encadeamento por ponteiros
- Complexidade temporal (O(n), O(1))
- Gerenciamento manual de memória
- Organização modular de código

---

## 🎯 Objetivo do Projeto

Consolidar os fundamentos de Estruturas de Dados e entender como elas funcionam internamente, sem uso de bibliotecas prontas (como STL).

---

## 📊 Complexidade das Operações

| Estrutura | Inserção | Remoção | Busca |
|-----------|----------|---------|--------|
| Lista     | O(1) ou O(n) | O(n) | O(n) |
| Pilha     | O(1) | O(1) | O(n) |
| Fila      | O(1) | O(1) | O(n) |

---

## 📚 Aprendizados

- Como estruturas são implementadas internamente
- Importância da organização lógica dos dados
- Cuidados com vazamento de memória
- Diferença entre abstração e implementação real
