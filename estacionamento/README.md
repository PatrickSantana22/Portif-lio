# 🚗 Sistema de Gerenciamento de Estacionamento

Sistema desenvolvido para simular o controle completo de um estacionamento, incluindo registro de veículos, cálculo de permanência e geração de cobrança automática.

---

## ⚙ Funcionalidades

- Registro de entrada de veículos
- Registro de saída
- Cálculo automático de tempo de permanência
- Cálculo de valor a pagar com base no tempo
- Controle de vagas disponíveis
- Validação de placa
- Listagem de veículos estacionados
