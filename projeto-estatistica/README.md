# 📊 Projeto de Estatística

Projeto acadêmico desenvolvido para aplicação prática de conceitos estatísticos, envolvendo análise de dados, interpretação de resultados e geração de relatórios.

---

## 📈 Conteúdos Aplicados

- Estatística descritiva
- Média, mediana e desvio padrão
- Distribuição de probabilidade
- Visualização de dados
- Interpretação estatística

---

## 🛠 Ferramentas Utilizadas

- Linguagem R / Quarto (.qmd)
- Modelagem matemática
- Geração de relatório automatizado

---

## 🎯 Objetivo

Aplicar conceitos estatísticos na análise de dados reais, desenvolvendo raciocínio analítico e interpretação quantitativa.

---

## 📚 Aprendizados

- Tratamento de dados
- Organização de resultados
- Construção de relatórios técnicos
- Interpretação estatística aplicada
