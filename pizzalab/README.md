# 🍕 PizzaLab

Aplicativo mobile desenvolvido em **Java no Android Studio**, simulando o sistema de pedidos de uma pizzaria com controle de clientes, endereço de entrega e sistema de fidelidade.

---

## 📱 Funcionalidades

- Cadastro de usuário
- Seleção de tamanho da pizza
- Escolha de sabores
- Adição de bebidas
- Tela de endereço de entrega
- Tela de resumo do pedido
- Sistema de contagem de pedidos
- Promoção automática: 1 pizza 47cm grátis a cada 10 pedidos

---

## 🛠 Tecnologias Utilizadas

- Java
- Android Studio
- XML (Layouts)
- SharedPreferences
- Lógica de contagem e persistência local

---

## 🧠 Conceitos Aplicados

- Manipulação de Activities
- Navegação entre telas
- Persistência de dados
- Estruturação de fluxo de pedido
- Lógica de regras de negócio

---

## 🎯 Objetivo do Projeto

Simular um aplicativo real de delivery, aplicando conceitos de desenvolvimento mobile e lógica de negócios.

---

## 📚 Aprendizados

- Estruturação completa de aplicativo Android
- Organização de código orientado a objetos
- Implementação de regras comerciais
- Gerenciamento de estado da aplicação
