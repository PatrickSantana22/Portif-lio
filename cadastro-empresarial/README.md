# 🏢 Sistema de Cadastro Empresarial

Projeto desenvolvido em **C, Java e Python**, com o objetivo de comparar diferentes formas de implementação de um mesmo sistema em múltiplas linguagens.

---

## 📋 Funcionalidades

- Cadastro de empresas
- Registro de CNPJ
- Armazenamento de dados
- Consulta de registros
- Atualização de informações
- Exclusão de cadastro

---

## 🛠 Linguagens Utilizadas

- C (programação estruturada)
- Java (orientação a objetos)
- Python (programação de alto nível)

---

## 🧠 Conceitos Aplicados

- Estruturação de dados
- Manipulação de arquivos
- Orientação a objetos
- Modularização de código
- Validação de dados

---

## 🎯 Objetivo

Comparar:

- Organização de código
- Complexidade de implementação
- Nível de abstração
- Produtividade entre linguagens

---

## 📚 Aprendizados

- Diferença entre paradigmas
- Gerenciamento de memória em C
- Abstração em Java
- Produtividade em Python
